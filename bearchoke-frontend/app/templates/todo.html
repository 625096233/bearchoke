ToDos with CQRS over WebSocket

See how using Spring, Spring Security, WebSocket, STOMP, AngularJS and the Axon CQRS framework makes the front and the back-end completely event driven.

This is the most advanced sample. It secures the WebSocket channel in a similar way to our REST controllers using our pre-authenticated token to create a SecureContext from which to handle user-centric message channels. The server leverages Axon's commandBus to dispatch commands in an event-driven way using the CQRS pattern.
